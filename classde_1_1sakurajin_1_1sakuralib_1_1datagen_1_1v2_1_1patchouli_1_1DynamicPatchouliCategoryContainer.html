Detailed Description

This class contains all data of a dynamic patchouli category.

This is used to add data to patchouli at runtime. Usually you would not call the constructor of this class directly but use PatchouliDataManager instead. You can add categories and entries to this container. When you add a category you get a new container for that category. You can then add entries to that category. This way you can create a tree of categories and entries.
