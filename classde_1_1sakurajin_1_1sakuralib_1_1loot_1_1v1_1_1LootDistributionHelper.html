This method returns a power distribution for the given order and element count.

The distribution is normalized and the sum of all elements is roughly 1. The distribution is cached, so it is not necessary to cache it yourself.

It is calculated using the following formula:

  • totalWeight = sum(i = 0, elementCount - 1, (elementCount - i) ^ order)
  • elementWeight(i) = (elementCount - i) ^ order / totalWeight The totalWeight is used to normalize the distribution.
