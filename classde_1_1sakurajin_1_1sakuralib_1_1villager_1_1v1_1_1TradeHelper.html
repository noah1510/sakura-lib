Detailed Description

A helper class to create trades for villagers.

It will automatically use currency instead of emeralds if numismatic-overhaul is present. The prices can either be set using emeralds or currency and will be converted automatically. The use of currency is recommended as it will automatically scale with the currency value and will automatically use emerald blocks if the price is higher than 64 emeralds. The registerTrades method can be used to register all trades for a profession at once using a list of TradeParameters.
